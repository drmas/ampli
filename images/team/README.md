# Team Images Folder

Add team member photos here with the following filenames:

## Required Images:

1. **attia.jpg** - Photo for Attia (@ATTIAMO)
2. **Nada.jpg** - Photo for Nada Elnadi (@NADAELNADI)
3. **Medha.jpg** - Photo for Medha (@MEDHA315)

## Image Specifications:

- **Format**: JPG or PNG
- **Dimensions**: Square aspect ratio (e.g., 500x500px or 800x800px)
- **Size**: Optimized for web (recommended < 200KB per image)
- **Content**: Professional headshots or profile photos
- **Quality**: High resolution but web-optimized

## Fallback Behavior:

If an image fails to load or doesn't exist, the avatar will automatically display the first letter of the person's name as a fallback.

## Adding New Team Members:

When adding new team members, add their photo here and update `/about/team/index.html` with their information.
